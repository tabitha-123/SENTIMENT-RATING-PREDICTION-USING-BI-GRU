# Sentiment-Aware Rating Prediction Using Bi-GRU

## About
This project implements a sentiment-aware rating 
prediction system for YouTube reviews using deep 
learning models (BERT, LSTM, Bi-GRU).

## Published Research
IEEE ICoECIT 2026 International Conference
DOI: 10.1109/ICoECIT68303.2026.11497399

## Author

M.Tech CSE - RGMCET, Nandyal

## Results
- Bi-GRU: 99% accuracy
- BERT: 51% accuracy  
- LSTM: 43% accuracy
